Curso JavaScript - Exercício 020

Exercícios

O propósito deste exercício é desenvolver um código que, ao clicar no botão, seja iniciado um prompt para o usuário inserir o nome do mês para que o resultado indique em qual estação do ano está o mês, o resultado deverá aparecer na tela. No exemplo sugerido, não são consideradas as datas, apenas os meses, logo, o retorno não será exatamente preciso, considerando que as estações do ano têm início antes do fim do mês da mudança.
