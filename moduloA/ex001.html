Hello World!

Os primeiros comandos abaixo interagem com o usuário de diferentes maneiras, seja informando ou requisitando dados e confirmações.
No entanto, os dados e respostas não serão armazenados até que seja definido para tal.

        
        window.alert("Minha primeira mensagem!") // Exibição de notas na janela incial.
        window.confirm("Está gostando das aulas?") // Mensagens de confirmação cuja resposta pode
        ser positiva ou negativa de acordo com a escolha do usuário
        window.prompt("Qual o seu nome?") // Solicitação de dados do usuário e/ou dados para
         armazenamento e uso em códigos de cálculos e cadastros, por exemplo.
