Utilizando Alerts

Aqui o prompt irá solicitar um dado do usuário através do prompt e retornará um alerta com o dado armazenado em uma variável.


    var nome = window.prompt("Qual o seu nome?")
    window.alert("É um grande prazer lhe conhecer, " + nome + "!")

Abaixo alguns comandos que podem ser utilizados para provocar interação com o usuário


    window.alert("Minha primeira mensagem!") // Exibição de notas na janela incial.
    window.confirm("Está gostando das aulas?") // Mensagens de confirmação cuja resposta pode ser positiva ou negativa de acordo com a escolha do usuário
    window.prompt("Qual o seu nome?") // Solicitação de dados do usuário e/ou dados para armazenamento e uso em códigos de cálculos e cadastros, por exemplo.
